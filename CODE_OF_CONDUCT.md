# Code of Conduct

## Our Standard

Contributors are expected to be direct, respectful, and technically honest.

Unacceptable behavior includes:

- harassment
- personal attacks
- discrimination
- deliberate disruption
- knowingly misleading technical claims

## Enforcement

Project maintainers may remove comments, close discussions, or block access for behavior that harms collaboration.

## Scope

This applies to issues, pull requests, discussions, and any other project communication channels.
